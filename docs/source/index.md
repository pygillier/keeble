# Keeble

A self-hostable, mobile-first family document vault. Publish guides,
checklists, and reference docs — router reboot steps, appliance manuals,
insurance contacts — so the rest of the family can self-serve.

Keeble is multi-tenant: each family's data is fully isolated within the same
deployment. Documents are written in Markdown and rendered as accordion-style
step guides.
