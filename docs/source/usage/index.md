# Usage Guide

Keeble accounts have one of two roles, defined per family member:

- **`editor`** — can create, edit, publish, and delete documents, and manage
  family members
- **`reader`** — can browse and search published documents only

This guide splits everyday usage into three perspectives:

```{toctree}
:maxdepth: 1

admin
writer
reader
```

```{note}
"Admin" and "Writer" below are **not** separate roles in Keeble — both
describe things an **editor** account can do. "Admin" covers the first-run
setup wizard and family member management; "Writer" covers day-to-day
document authoring. Any editor account can do both. "Reader" describes the
separate, restricted `reader` role.
```
