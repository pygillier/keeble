# Creating & Editing Documents

The editor is designed for desktop use and gives you full control over your guides.

## Creating a New Document

1. Click **New document** in the sidebar or header.
2. Enter a **title** — Keeble will generate a URL slug automatically.
3. Write your content in the **Markdown editor** (left pane).
4. The **live preview** (right pane) updates as you type.
5. Click **Save** when you are done.

## Markdown Basics

Keeble documents are written in Markdown. Here are the most useful formatting shortcuts:

```markdown
# Heading 1
## Heading 2
### Heading 3

**Bold text**
*Italic text*

- Bullet list item
- Another item

1. Numbered step
2. Another step

> Tip or warning block
```

## Document Properties (Sidebar)

The right sidebar (or collapsible panel on smaller screens) exposes:

| Field | Description |
|---|---|
| **Title** | Displayed at the top of the guide |
| **Slug** | URL path (auto-generated, editable) |
| **Tags** | Select existing tags or create new ones |

## Uploading Images

1. In the editor, click the **image upload** button in the toolbar, or drag a file directly onto the editor.
2. Keeble uploads the file to PocketBase and inserts a Markdown image reference at the cursor position.
3. Supported formats: JPEG, PNG, GIF, WebP.

> Images are stored with the document in PocketBase. Deleting a document also removes its associated images.

## Importing a Markdown File

You can import an existing `.md` file instead of writing from scratch.

See the [Importing Documents](/guide/importing) page for details.

## Deleting a Document

Open the document, click the **⋯** (more options) menu, then select **Delete**. You will be asked to confirm.
