# Tags

Tags help you organise documents into categories so family members can browse by topic.

## What Tags Are For

Instead of remembering the exact name of a guide, a family member can browse to the **Tags** page and pick a category like `router`, `washing machine`, or `printer` to see all related guides.

## Creating Tags

Tags are created automatically when you type a new tag name in the document editor's tag field. You can also manage tags directly from the **Tags** page (admin only).

Each tag has:
- A **name** (e.g. `router`)
- A **colour** (hex value, used for the tag badge)

## Assigning Tags to a Document

1. Open the document in the editor.
2. In the **Tags** field (right sidebar), type to search for an existing tag or enter a new name.
3. Select from the dropdown or press **Enter** to create.
4. Save the document.

A document can have multiple tags.

## Managing Tags

Admins can rename, recolour, or delete tags from the Tags management page:

1. Navigate to **Tags** in the top navigation.
2. Click the **Manage** button (visible to admins only).
3. Edit or delete tags as needed.

> Deleting a tag removes it from all documents that use it — the documents themselves are not deleted.
