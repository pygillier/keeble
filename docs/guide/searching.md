# Searching for Documents

The search bar is the fastest way to find a guide on Keeble.

## Using the Search Bar

1. Tap or click the **search bar** at the top of the page.
2. Start typing a word — results appear immediately as you type.
3. Tap a result to open the guide.

Keeble searches across:

- Document **titles** (highest priority)
- **Tags** attached to the document
- The document **body** text

> Results are ranked so that title matches appear first, followed by tag matches, then body matches.

## Browsing by Tag

If you prefer to browse rather than search, use the **Tags** page (bottom navigation bar → Tags icon).

Select a tag to see all guides in that category, for example:
- `network` → Wi-Fi, router, and internet guides
- `appliance` → washing machine, dishwasher, etc.

## Tips

- Short keywords work best: type `router` instead of `how do I reboot my router`.
- On mobile, the search bar stays fixed at the top while you scroll through results.
