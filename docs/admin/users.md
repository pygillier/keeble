# Managing Users

## User Roles

Keeble currently has two roles:

| Role | Capabilities |
|---|---|
| **Admin** | Read, create, edit, delete documents; manage tags and settings; manage users |
| **Member** | Read documents and search only |

## Inviting Users

If **Allow registration** is enabled in Settings, users can sign up themselves at `/register`.

To invite someone manually:

1. Go to **Settings → Users**.
2. Click **Invite user**.
3. Enter their email address — they will receive a link to set their password.

## Disabling Registration

To prevent new sign-ups:

1. Go to **Settings → General**.
2. Toggle **Allow registration** off.
3. Save.

Existing accounts are not affected. New users can only be added by an admin.

## Removing a User

1. Go to **Settings → Users**.
2. Find the user and click **Remove**.
3. Confirm the action.

Removed users cannot log in, but their documents and contributions remain intact.
